Mind Map

ภาพรวม If Clauses

มองเห็นความสัมพันธ์ของประโยคเงื่อนไขทั้ง 4 ประเภทในภาพเดียว

Structure

โครงสร้าง ทั้ง 4 ประเภท

ทำความเข้าใจโครงสร้างและความหมายของแต่ละ conditional

Zero Conditional

ความจริงทั่วไป / กฎธรรมชาติ

ใช้พูดถึงสิ่งที่เป็นความจริงเสมอ ผลลัพธ์ที่เกิดขึ้นแน่นอนเมื่อมีเงื่อนไขนั้น

If + Subject + V1, Subject + V1
ตัวอย่าง: If you heat ice, it melts.
First Conditional

เป็นไปได้จริงในอนาคต

ใช้พูดถึงสถานการณ์ที่อาจเกิดขึ้นได้จริงในอนาคต มีความเป็นไปได้สูง

If + Subject + V1, Subject + will + V1
ตัวอย่าง: If it rains, I will stay home.
Second Conditional

สมมติ / ไม่น่าเป็นไปได้ (ปัจจุบัน)

สมมติสถานการณ์ที่ไม่เป็นจริง หรือแทบไม่มีทางเกิดขึ้นในปัจจุบัน

If + Subject + V2, Subject + would + V1
ตัวอย่าง: If I were rich, I would travel the world.
Third Conditional

สมมติอดีต / เสียดายที่ผ่านไปแล้ว

สมมติว่าถ้าเหตุการณ์ในอดีตเกิดต่างออกไป ผลลัพธ์จะเป็นอย่างไร

If + Subject + had + V3, Subject + would have + V3
ตัวอย่าง: If I had studied, I would have passed.

📊 ตารางสรุปโครงสร้าง

TypeIf ClauseMain Clauseการใช้
Zero If + V1 V1 ความจริงสากล
First If + V1 will + V1 อนาคตที่เป็นไปได้
Second If + V2 / were would + V1 สมมติปัจจุบัน
Third If + had + V3 would have + V3 สมมติอดีต
How to Use

วิธีเลือกใช้ อย่างถูกต้อง

คำถาม 4 ข้อนี้จะช่วยให้คุณเลือก conditional ได้ถูกต้อง

🌍

ความจริงทั่วไปหรือไม่?

ถ้าพูดถึงกฎธรรมชาติ หรือสิ่งที่เป็นจริงเสมอ → ใช้ Zero Conditional
"If you drop a ball, it falls."

🔮

อนาคต เป็นไปได้จริงไหม?

ถ้าสถานการณ์มีโอกาสเกิดขึ้นจริงในอนาคต → ใช้ First Conditional
"If I have time, I will help you."

💭

สมมติ ไม่เป็นความจริงตอนนี้?

ถ้าเป็นเรื่องสมมติที่ไม่น่าเกิดขึ้น → ใช้ Second Conditional
"If I were a bird, I would fly."

⏰

สมมติอดีต ที่ผ่านไปแล้ว?

ถ้าสมมติเหตุการณ์อดีตที่เปลี่ยนไม่ได้ → ใช้ Third Conditional
"If I had known, I would have called."

💡
Tips: ใน Second Conditional ใช้ "were" แทน "was" ทุก subject เช่น "If I were you…" แม้กับ I/He/She ก็ตาม เพราะเป็นรูป Subjunctive Mood
Examples

ตัวอย่าง ประโยคทุกรูปแบบ

ประโยคบอกเล่า ปฏิเสธ และคำถาม ของทั้ง 4 conditional

Zero Conditional
If you heat water to 100°C, it boils.
🇹🇭 ถ้าคุณให้ความร้อนน้ำถึง 100°C มันจะเดือด (ความจริงวิทยาศาสตร์)
If dogs eat chocolate, they get sick.
🇹🇭 ถ้าสุนัขกินช็อกโกแลต มันจะป่วย
First Conditional
If it rains tomorrow, I will stay home.
🇹🇭 ถ้าฝนตกพรุ่งนี้ ฉันจะอยู่บ้าน
If she studies hard, she will pass the exam.
🇹🇭 ถ้าเธอเรียนหนัก เธอจะสอบผ่าน
Second Conditional
If I were rich, I would travel the world.
🇹🇭 ถ้าฉันรวย ฉันคงจะท่องเที่ยวรอบโลก (แต่จริงๆ ฉันไม่รวย)
If he had a car, he would drive to work.
🇹🇭 ถ้าเขามีรถ เขาคงขับรถไปทำงาน
Third Conditional
If I had studied, I would have passed the test.
🇹🇭 ถ้าฉันได้ตั้งใจเรียน ฉันคงสอบผ่านไปแล้ว (แต่ไม่ได้ทำ)
If she had left earlier, she would have caught the bus.
🇹🇭 ถ้าเธอออกไปก่อนหน้านี้ เธอคงทันรถบัสแล้ว
Zero Conditional
If you don't water plants, they don't grow.
🇹🇭 ถ้าคุณไม่รดน้ำต้นไม้ มันก็จะไม่เติบโต
First Conditional
If you don't hurry, you will not catch the train.
🇹🇭 ถ้าคุณไม่รีบ คุณจะไม่ทันรถไฟ
If I don't sleep early, I won't wake up on time.
🇹🇭 ถ้าฉันไม่นอนแต่เนิ่นๆ ฉันจะตื่นไม่ทันเวลา
Second Conditional
If I didn't have a job, I wouldn't be able to pay rent.
🇹🇭 ถ้าฉันไม่มีงาน ฉันคงไม่สามารถจ่ายค่าเช่าได้
Third Conditional
If he hadn't been late, he wouldn't have missed the meeting.
🇹🇭 ถ้าเขาไม่มาสาย เขาคงไม่พลาดการประชุม
If we hadn't forgotten the map, we wouldn't have gotten lost.
🇹🇭 ถ้าเราไม่ลืมแผนที่ เราคงไม่หลงทาง
ℹ️
Note: ในประโยคคำถาม ส่วน If clause ไม่เปลี่ยน แต่ main clause เปลี่ยนเป็น question form
Zero Conditional
What happens if you mix red and blue?
🇹🇭 จะเกิดอะไรขึ้นถ้าคุณผสมสีแดงกับสีน้ำเงิน?
First Conditional
Will you come to the party if I invite you?
🇹🇭 คุณจะมางานปาร์ตี้ไหมถ้าฉันเชิญ?
What will you do if you lose your phone?
🇹🇭 คุณจะทำอย่างไรถ้าคุณทำโทรศัพท์หาย?
Second Conditional
What would you do if you won the lottery?
🇹🇭 คุณจะทำอะไรถ้าคุณถูกหวย?
Where would you go if you could travel anywhere?
🇹🇭 คุณจะไปที่ไหนถ้าคุณสามารถเดินทางได้ทุกที่?
Third Conditional
Would you have stayed if you had known about the problem?
🇹🇭 คุณจะอยู่ต่อไหมถ้าคุณรู้เกี่ยวกับปัญหานั้น?
What would have happened if they hadn't arrived in time?
🇹🇭 จะเกิดอะไรขึ้นถ้าพวกเขาไม่มาทัน?

Quiz Time

แบบทดสอบ If Clauses

20 ข้อ (If Clauses 10 ข้อ + Synonyms 10 ข้อ) เลือกคำตอบที่ถูกต้อง มีคำอธิบายทุกข้อ

ข้อที่ 1 / 20
If you ______ water, it freezes.
เลือกกริยาที่ถูกต้องสำหรับ Zero Conditional
✅ ถูกต้อง! ตอบ D: cool — Zero Conditional ใช้ If + V1, V1 (present simple ทั้งสองส่วน) เพราะพูดถึงกฎธรรมชาติ: "If you cool water (below 0°C), it freezes."
ข้อที่ 2 / 20
If she ______ harder, she will get a promotion.
First Conditional — เลือกรูปกริยาที่ถูก
✅ ถูกต้อง! ตอบ B: works — First Conditional ใช้ If + V1 (present simple) ตามด้วย will + V1 เพราะเป็นอนาคตที่มีโอกาสเกิดขึ้นจริง
ข้อที่ 3 / 20
If I ______ a superpower, I would fly.
Second Conditional — สมมติที่ไม่เป็นจริง
✅ ถูกต้อง! ตอบ C: had — Second Conditional ใช้ If + V2 (past simple) + would + V1 เป็นการสมมติสิ่งที่ไม่เป็นจริงในปัจจุบัน
ข้อที่ 4 / 20
If they had arrived earlier, they ______ the concert.
Third Conditional — เลือก main clause ที่ถูก
✅ ถูกต้อง! ตอบ C: would have caught — Third Conditional ใช้ If + had + V3 ตามด้วย would have + V3 เป็นการสมมติอดีตที่เปลี่ยนไม่ได้
ข้อที่ 5 / 20
เลือกประโยค Second Conditional ที่ถูกต้อง
✅ ถูกต้อง! ตอบ C — Second Conditional ที่ถูกต้องใช้ "were" (Subjunctive Mood) กับทุก subject รวมถึง I ด้วย และใช้ would (ไม่ใช่ will) ใน main clause
ข้อที่ 6 / 20
What type of conditional is this?
"If you press this button, the alarm rings."
✅ ถูกต้อง! ตอบ A: Zero Conditional — ใช้ V1 ทั้งสองส่วน (press, rings) เป็นการบอกผลที่เกิดขึ้นเสมอเมื่อทำสิ่งนั้น เป็นเหตุ-ผลที่แน่นอน
ข้อที่ 7 / 20
If the weather ______ nice this weekend, we will have a picnic.
เลือกกริยาที่เหมาะสม
✅ ถูกต้อง! ตอบ B: is — เนื่องจาก main clause ใช้ "will have" จึงเป็น First Conditional และ If clause ต้องใช้ V1 (present simple) คือ "is"
ข้อที่ 8 / 20
ประโยค Third Conditional แบบปฏิเสธที่ถูกต้องคือข้อใด?
✅ ถูกต้อง! ตอบ D — Third Conditional ปฏิเสธใช้ "If + hadn't + V3, wouldn't have + V3" — A คือ Second Conditional, B ผิด grammar, C ผิดโครงสร้าง
ข้อที่ 9 / 20
"What ______ you do if you found a wallet on the street?"
คำถาม Second Conditional
✅ ถูกต้อง! ตอบ C: would — เป็นประโยคคำถาม Second Conditional (If + V2, would + V1) เพราะ "found" เป็น past tense ใน if clause จึง main clause ต้องใช้ would
ข้อที่ 10 / 20
เลือกประโยคที่ใช้ conditional ได้ถูกต้องที่สุด
✅ ถูกต้อง! ตอบ B — First Conditional ที่ถูกต้อง: If + V1 (see), will + V1 (tell) — A ผิดเพราะใช้ will ใน if clause, C ผิด tense, D ผิด conditional type
📚 Synonym Quiz
ข้อ 11–20: เลือก synonym ที่ถูกต้องของแต่ละคำ
ข้อที่ 11 / 20
คำใดเป็น Synonym ของ "Happy"?
เลือกคำที่มีความหมายใกล้เคียงที่สุด
✅ ถูกต้อง! ตอบ B: Joyful — "Joyful" แปลว่า เปี่ยมสุข มีความหมายเหมือน Happy เพราะทั้งคู่หมายถึงความรู้สึกมีความสุข ส่วนคำอื่นมีความหมายต่างออกไปโดยสิ้นเชิง
ข้อที่ 12 / 20
คำใด ไม่ใช่ Synonym ของ "Big"?
✅ ถูกต้อง! ตอบ D: Tiny — "Tiny" แปลว่า จิ๋ว/เล็กมาก ซึ่งเป็น antonym (คำตรงข้าม) ของ Big ไม่ใช่ synonym ส่วน Huge, Enormous, Vast ล้วนหมายถึงขนาดใหญ่เหมือนกัน

"She is very intelligent." คำที่ขีดเส้นใต้ มี synonym คือข้อใด?
✅ ถูกต้อง! ตอบ C: Brilliant — "Intelligent" และ "Brilliant" ต่างหมายถึงความฉลาดเก่งกาจ ทั้งคู่ใช้แทนกันได้ในบริบทส่วนใหญ่ คำอื่นมีความหมายที่ต่างออกไป
ข้อที่ 14 / 20
เลือก synonym ของ "Difficult" ที่เหมาะสมที่สุด
✅ ถูกต้อง! ตอบ A: Challenging — "Challenging" และ "Difficult" ต่างหมายถึงสิ่งที่ต้องใช้ความพยายามมาก ส่วน Simple, Pleasant, Smooth ล้วนมีความหมายในเชิงบวกหรือง่าย ซึ่งตรงข้ามกัน
ข้อที่ 15 / 20
คำว่า "Obtain" เป็น synonym ของคำใด?
✅ ถูกต้อง! ตอบ C: Get — "Obtain" แปลว่า ได้รับ/หามาได้ มีความหมายเดียวกับ "Get" ในบริบทของการได้รับบางสิ่ง เช่น "I obtained / got good results." ส่วนคำอื่นมีความหมายตรงข้าม
ข้อที่ 16 / 20
"The police ______ the protest." เลือกคำที่เป็น synonym ของ "show/display"
✅ ถูกต้อง! ตอบ B: demonstrated — "Demonstrate" เป็น synonym ของ "show/display" แปลว่า สาธิต/แสดงให้เห็น ในประโยคนี้หมายถึงตำรวจสลายการประท้วง (เดิมคำนี้มีความหมายว่า แสดงออก/ประท้วง ด้วย)
ข้อที่ 17 / 20
คำว่า "Gorgeous" เป็น synonym ของคำใด?
✅ ถูกต้อง! ตอบ D: Beautiful — "Gorgeous" หมายถึง สวยงามมาก ตระการตา เป็น synonym ของ "Beautiful" โดย Gorgeous มีระดับความเข้มข้นสูงกว่า เหมือนกับการพูดว่า "สวยสุดๆ"
ข้อที่ 18 / 20
ข้อใดคือคู่ synonym ที่ถูกต้อง?
✅ ถูกต้อง! ตอบ C: Change – Alter — ทั้งสองคำมีความหมายว่า เปลี่ยนแปลง/ดัดแปลง จึงเป็น synonym กัน ส่วนข้ออื่นล้วนเป็นคู่ antonym (ความหมายตรงข้าม)
ข้อที่ 19 / 20
"She ______ by the window for a long time." เลือก synonym ของ "walked slowly/wandered"
✅ ถูกต้อง! ตอบ B: strolled — "Stroll" เป็น synonym ของ "walk" ในความหมายว่า เดินเล่นช้าๆ อย่างผ่อนคลาย "Sprint" = วิ่งเร็ว, "Jump" = กระโดด, "Crawl" = คลาน ซึ่งต่างจากการเดิน
ข้อที่ 20 / 20
คำว่า "Affection" เป็น synonym ของคำใดในประโยคนี้?
"He showed great ______ for his family."
✅ ถูกต้อง! ตอบ B: Love — "Affection" แปลว่า ความรักใคร่ผูกพัน เป็น synonym ของ "Love" ความหมายใกล้เคียงกันมาก แต่ Affection มักหมายถึงความรักที่อ่อนโยนและความผูกพัน ไม่ใช่ความรุนแรงทางอารมณ์
